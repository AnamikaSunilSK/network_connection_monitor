# network_connection_monitor

Network Connection Monitor is a real-time network surveillance tool that displays all active TCP/UDP connections on your Windows system, including local/foreign addresses, connection states (ESTABLISHED, LISTENING, TIME_WAIT), and process IDs (PIDs). Built with Python's tkinter and Windows netstat, this tool helps detect malware callbacks, unauthorized outbound connections, hidden backdoors, and suspicious listening ports. Features include live auto-refresh every 3 seconds, smart filtering by connection state, search functionality, and exportable reports for forensic analysis. Perfect for security analysts, incident responders, system administrators, and anyone concerned about malicious network activity on their machine.
